## 📚 Sobre o Projeto

O **Jogo do Glécio** é uma adaptação digital de uma dinâmica criada pelo professor Glécio no ensino médio. A versão original consistia em desafios orais com a tabuada, onde os alunos anotavam as respostas para correção posterior. Este projeto leva essa prática para o mundo digital, tornando-a acessível a qualquer estudante, em qualquer lugar, a qualquer hora.

## 🛠️ Como Rodar Localmente

```bash
# Clone o repositório
git clone https://github.com/mateus-f/jogo-do-glecio.git
cd jogo-do-glecio

# Instale as dependências
npm install

# Inicie o servidor local
npm run dev
```
